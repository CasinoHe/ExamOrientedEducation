# Agent-04 Student Frontend Brief

This agent exists to define the student-facing experience for studying, practice, and review on browser-based devices.

## Your Mission

Define the student-facing UX, page map, lesson/review/practice flows, and mobile-first browser behavior for the MVP. Your work should make the product usable on iPhone and iPad while staying aligned with the History learning model.
